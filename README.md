# Dream Dungeon Design Bible

Hand-authored starter wiki for the Dream Dungeon design documents and reference data.

## Local preview

1. Serve this directory with any static web server, for example `npx serve .`.
2. Open the local address shown by the server.

The site is intentionally plain HTML, CSS, and JavaScript. Add or revise wiki entries directly in `index.html` as the project develops.

## GitHub Pages

The included workflow deploys the repository root to GitHub Pages whenever changes are pushed to `main`. In the GitHub repository, open **Settings → Pages** and set **Source** to **GitHub Actions** once.
